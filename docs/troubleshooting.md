# Troubleshooting

## Docker networking

Problem:

Address already in use

Solution:

The Docker bridge network was recreated.

---

## Routing

Problem:

Host A could not reach Host B.

Solution:

The default gateway of Host A was updated to Gateway A.

---

## StrongSwan

Problem:

Tunnel remained in CONNECTING state.

Solution:

StrongSwan was started on Gateway B.
